Concept Core

What to look for and why it works

When to use it

  • Sorted array or sortable data.
  • Pair or triplet target condition.
  • Need O(1) extra space behavior.

Invariant

At each step, you remove one impossible region. If sum is too small, only increasing the left value can help. If too large, only decreasing the right value can help.

Complexity memory hook

Each pointer moves in one direction and never rewinds. That means at most n moves per pointer: O(n).

Interview Script

What to say before coding

"The input is sorted, so pointer movement is monotonic and meaningful. I will start one pointer at each end. If the sum is too small, I move left forward to increase sum. If too large, I move right backward to decrease sum. Each pointer moves at most n times, so total time is O(n) with O(1) extra space."
