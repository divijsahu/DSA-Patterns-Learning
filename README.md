# DSA Pattern Learning

Premium study material for mastering interview patterns, organized as a deliberate curriculum rather than a loose collection of notes.

This repository teaches the 30 most useful DSA patterns through the same learning loop used by strong interview candidates:

1. Recognize the clue.
2. Understand why the pattern works.
3. Learn the canonical template.
4. Practice representative problems.
5. Review the mistakes that actually matter.

## How To Use This Repo

Read it like a course, not a reference manual.

1. Start with Part 1 and do not skip the pattern fingerprints.
2. For each pattern, read the intuition, then the template, then the traps.
3. Solve a few curated problems before moving on.
4. Keep a separate notebook for the clue that helped you identify each pattern.
5. Revisit the master cheat sheet in Part 3 after every practice session.

## Suggested Study Loop

- Read one pattern end to end.
- Rewrite the template from memory.
- Solve 2 to 4 problems using that pattern.
- Record the exact trigger words or structural clues that pointed to the pattern.
- Return to the same pattern a few days later and test recall without looking.

## Editorial Standard

Each pattern in this repository is written to answer four questions clearly:

- When should I use it?
- Why does it work?
- What does the implementation look like?
- What breaks most often in interviews?

That structure is what makes the repo useful as premium study material instead of just a collection of solutions.

## Recommended Path

If you are studying from zero, use this order:

1. Foundation Patterns
2. Intermediate Patterns
3. Advanced Patterns
4. Master Cheat Sheet
5. Mixed practice and timed recall

If you want to turn this into a long-term interview system, keep the pattern index open while practicing and annotate it with your own problem notes.
